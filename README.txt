Программа позволяет зашифровать/дешифровать текстовое сообщение шифром Первой Мировой Войны или шифром Цезаря.
В качестве первого шифра реализован способ, описанный на сайте по адресу
https://www.e-reading.club/chapter.php/1043017/113/Malashkina_-_Ya_poznayu_mir._Kriminalistika.html
Его алфавит представлен значениями буква - код буквы, считывается из json файла, при необходимости можно легко поменять язык, поменяв буквы в файле.
Алфавит Цезаря реализован лишь для русского языка, нет возможности поменять язык без изменения исходного кода.
